# Gemini CLI Learning Repository

This repository serves as a comprehensive guide and collection of content related to using and implementing the Gemini CLI. It's structured into subfolders, each focusing on a specific aspect of the Gemini CLI, from installation to advanced usage and custom server setups.

## Table of Contents

| Folder | Description |
| :------------------------------ | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [01_Installation_Guide](01_Installation_Guide/README.md) | This folder contains notes and steps related to the installation and setup of the Gemini CLI. It covers prerequisites, installation commands, and initial configuration to get the CLI up and running. |
| [03_Github_MCP_Server](03_Github_MCP_Server/README.md) | This project involves a Python script to modify a CSV file, specifically updating 'Sale price' values, and demonstrates concepts related to GitHub repository management, though some steps were initially uncompleted due to environment limitations. |
| [04_Custom_Github_MCP_Server](04_Custom_Github_MCP_Server/README.md) | This folder contains a `Dockerfile` for setting up a custom environment, potentially for GitHub MCP (Mission Control Platform) server or related services. It outlines the steps to containerize an application or environment, ensuring consistency and portability. |
